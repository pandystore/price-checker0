# نظام فاحص الأسعار (Price Checker)

تطبيق ويب لتفقد أسعار المنتجات وعرض الخصومات، مصمم للشاشات الذكية والأجهزة المحمولة داخل المتاجر والشركات.

## 🚀 المميزات
* **مسح الباركود:** يدعم الأجهزة اليدوية (Barcode Scanners) أو الكاميرا المباشرة.
* **إدارة الخصومات:** دعم الخصومات المباشرة وخصومات المجموعات مع تقريب الأسعار لأقرب 5 جنيهات.
* **شريط العروض:** شريط متحرك أسفل/أعلى الشاشة لعرض الإعلانات والتنبيهات.
* **لوحة تحكم:** محمية بكلمة مرور لتعديل الإعدادات، رفع خلفية، ولوجو خاص بالمتجر.
* **رفع البيانات عبر Excel:** استيراد أصناف المتجر مباشرة من شيت إكسيل.
* **مزامنة سحابية:** ربط وتحديث البيانات تلقائيًا عبر **GitHub API**.

---

## 📊 هيكل ملف الـ Excel
عند رفع ملف الأسعار، يجب أن تحتوي الأعمدة على الترتيب التالي:

| العمود | البيان |
| :--- | :--- |
| **A** | كود الصنف (Item Code) |
| **B** | كود التاجر (Merchant Code) |
| **C** | اسم الصنف (Product Name) |
| **D** | الوحدة (Unit) |
| **E** | سعر البيع (Selling Price) |
| **F** | السعر قبل الخصم (Original Price) |
| **G** | مجموعة الصنف (Group) |

---

## ⚙️ طريقة الربط بـ GitHub
1. أنشئ حسابًا على GitHub وحفظ البيانات في مستودع (Repository) جديد.
2. أنشئ **Personal Access Token** من إعدادات الحساب مع صلاحية `repo`.
3. افتح لوحة التحكم في التطبيق وأدخل:
   * **GitHub User**
   * **GitHub Repo**
   * **GitHub Token**
4. اضغط **حفظ** ثم **نشر التغييرات للجميع** لمزامنة ملف `data.json`.
