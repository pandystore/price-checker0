# 🏷️ كاشف الأسعار

تطبيق عرض أسعار المنتجات عن طريق مسح QR Code أو كتابة الكود.

---

## 📁 الملفات

| الملف | الوصف |
|-------|-------|
| `index.html` | التطبيق الرئيسي |
| `data.json` | ملف البيانات (المنتجات والعروض والإعدادات) |

---

## 🚀 طريقة التشغيل

### 1️⃣ رفع الملفات على GitHub Pages

1. أنشئ Repository جديد على [GitHub](https://github.com)
2. ارفع ملف **`index.html`** وملف **`data.json`** في الـ root
3. فعّل GitHub Pages:
   - Settings → Pages
   - Source: **Deploy from a branch**
   - Branch: **main** + **/ (root)**
4. افتح الرابط: `https://YOUR-USERNAME.github.io/REPO-NAME/`

### 2️⃣ من السيرفر المحلي

```bash
npx serve .
# أو
python -m http.server 8000
```

---

## ⚙️ تحديث البيانات

### الطريقة السهلة:

1. حرّر ملف **`data.json`** في جهازك
2. استبدل الملف القديم على GitHub بالملف الجديد
3. **كل المستخدمين هيشوفوا البيانات المحدثة تلقائيًا!**

### من لوحة التحكم:

1. افتح التطبيق ← اضغط ⚙️
2. أدخل كلمة المرور: **`1234`**
3. حرّر المنتجات / العروض / الألوان
4. اضغط **"تحميل data.json"**
5. حرّر الملف في جهازك ثم ارفعه على GitHub

## 🔑 لوحة التحكم

- افتح التطبيق ← اضغط ⚙️ (أسفل اليمين)

### التابويات:
- 📋 **البيانات**: عرض وإدارة المنتجات
- 🎨 **المظهر**: تغيير اسم المحل واللوجو والألوان
